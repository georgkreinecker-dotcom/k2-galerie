# Kommunikations-Dokumente – Struktur & was noch fehlt

**Stand:** 08.03.26  
**Zweck:** Eine klare Übersicht: Welches Dokument wofür, was ist fertig, was muss ausgefeilt werden. Gezielt und mit Struktur – damit du (Georg) siehst, wo du ansetzen kannst und wo noch Probleme liegen.

---

## Sportwagenmodus (verbindlich)

**Ein Zweck = ein Dokument, eine Quelle.** Kein Doppeltes, keine Sonderfälle.

| Regel | Konkret |
|-------|---------|
| **Ein Ablauf pro Anlass** | Presse = immer PRESSEARBEIT-STANDARD + MEDIENVERTEILER + Vorlage §6. Nicht „mal so, mal so“. |
| **Eine Vorlage pro Zweck** | Ansprache Künstler:in/Verein = eine Vorlage. Flyer = eine Vorlage. E-Mail nach Kauf = eine Vorlage. Keine zweite Variante ohne klaren Grund. |
| **Eine Ablage** | Pro Dokument ein fester Ort (docs/ oder mök2); alle nutzen dieselbe Quelle. |
| **Mehrsprachigkeit** | Ist **mitgeplant** (Struktur so, dass später pro Vorlage eine zweite Sprachversion dazukommt). **Umsetzung erst, wenn in Deutsch alles läuft.** Siehe docs/MEHRSPRACHIGKEIT-PLAN.md. |

## 2. Wo typischerweise Probleme entstehen (Checkliste für dich)

| Problem | Lösung / wo es steht |
|--------|------------------------|
| **„Welches Dokument nehme ich wann?“** | Diese Tabelle (§1): Zweck → ein Dokument. Presse = immer PRESSEARBEIT-STANDARD + MEDIENVERTEILER + Vorlage §6. |
| **„Ist der Text wirklich fertig zum Abschicken?“** | Presse §6: Nur [Platzhalter] ersetzen, dann versenden. Wenn du willst, können wir die Vorlage noch einmal redigieren (ein Satz „fertige Fassung“ pro Anlass). |
| **„An wen schicke ich was – und in welcher Reihenfolge?“** | MEDIENVERTEILER: zuerst regional (OÖN, MeinBezirk, Tips, ORF OÖ), dann überregional. PRESSEARBEIT-STANDARD Schritt 5. |
| **„Ich will einen Künstler / Verein gezielt ansprechen – was schreibe ich?“** | Dafür fehlt noch eine **gezielte Vorlage** (siehe §3 unten). Soll als nächstes ausgearbeitet werden. |
| **„Flyer für die Galerie / für Messen – was drauf?“** | Dafür fehlt noch ein **Flyer-/Handout-Muster** (eine Seite: Botschaft, QR, Kontakt) – dann druckfertig. |
| **„Nach dem Lizenzkauf: Was mailen wir?“** | E-Mail-Vorlagen (Danke, Link, nächster Schritt) fehlen noch – können wir als feste Texte anlegen, die du nur noch personalisierst. |
| **„Mehrsprachig – Presse/Ansprache auf Englisch?“** | Siehe docs/MEHRSPRACHIGKEIT-PLAN.md: klären, wo DE reicht und wo EN (oder andere Sprache) dazukommt; dann eine Quelle pro Sprache. |

## 3. Nächste Schritte – was wir gezielt ausarbeiten

**Reihenfolge (Vorschlag):**

1. **Ansprache Künstler:in / Verein (E-Mail oder Kurztext)**  
   - Ein Standardtext: Einladung zur Demo (ök2/VK2), Link, ein Satz warum K2. Du fügst nur Namen/Anrede ein.  
   - Ablage: z. B. docs/KOMMUNIKATION-VORLAGE-ANSPRACHE-KUENSTLER-VEREIN.md oder als Block in MEDIENSTUDIO-K2.

2. **Flyer / Handout (eine Seite, druckfertig)**  
   - Inhalt: Was ist K2 Galerie / ök2 / VK2 (2–3 Sätze), QR/Link, Kontakt.  
   - Ablage: Vorlage in docs oder mök2; Design/PDF so, dass du drucken oder drucken lassen kannst.

3. **E-Mail-Vorlagen (nach Lizenzkauf, Einladung)**  
   - Kurze, verbindliche Texte: „Danke für Ihren Kauf – hier Ihr Link …“, „Einladung zum Test …“.  
   - Ablage: eine Datei (z. B. docs/KOMMUNIKATION-EMAIL-VORLAGEN.md) oder Sektion in mök2; nur noch [Name], [Link] ersetzen.

4. **Presse-Vorlage §6 redigieren (optional)**  
   - Wenn du möchtest: eine „fertige Fassung“ pro Anlass (z. B. nur Eröffnung), ohne Platzhalter im Fließtext – dann wirklich copy-paste und nur Datum/Ort oben einsetzen.

5. **Mehrsprachigkeit Kommunikation**  
   - **Mitgeplant:** Struktur so, dass pro Vorlage später eine zweite Sprachversion (z. B. EN) dazukommt – eine Quelle pro Sprache. **Umsetzung erst, wenn in Deutsch alles läuft.** Siehe MEHRSPRACHIGKEIT-PLAN.md.
