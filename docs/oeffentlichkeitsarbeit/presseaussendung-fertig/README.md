# Presseaussendung – fertige Texte (Repo-Ablage)

**Ein Ort** für alles, was **fertig** ist: **versendet** oder **zur Weitergabe freigegeben** – Volltexte zum Kopieren (E-Mail, Nachricht), **nicht** nur Generator-Vorschau aus der App.

| | |
|---|---|
| **Hier** | Markdown (`.md`) – im Editor oder Cursor öffnen, Text markieren, in Mailprogramm einfügen |
| **App** | Admin → Event- und Medienplanung erzeugt **Entwürfe** aus Stammdaten; **dieser Ordner** = festgehaltene Endfassungen |

## Neue Datei anlegen

1. Namensvorschlag: `PRESSE-AUSSENDUNG-ARCHIV-YYYY-MM-DD-KURZ.md` (Aussendung) bzw. `PRESSE-ERINNERUNG-…-YYYY-MM-DD.md` (Folge).  
2. Oben: **Versanddatum, Von, An, Betreff** – wie bei den bestehenden Archiven.  
3. In [00-INDEX.md](00-INDEX.md) und ggf. in [../00-INDEX.md](../00-INDEX.md) eintragen.
