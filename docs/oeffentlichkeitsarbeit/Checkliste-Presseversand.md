# Checkliste – vor dem Presseversand

**Stand:** 17.03.26 · **Zweck:** Nichts vergessen vor dem Versand. Abhaken und los.

---

## Vor dem Versand

- [ ] **Text** aus Vorlage oder aus App (Presse-Paket kopieren); alle Platzhalter [Datum], [Ort], [Adresse], [Kontakt] ersetzt.
- [ ] **Betreff** im Format: „Presseinformation – [Anlass], [Datum], [Ort]“ (z. B. „Presseinformation – Eröffnung K2 Galerie, 24.04.26, Eferding“).
- [ ] **Verteiler:** E-Mail-Adressen aus [Medienverteiler-Kurz.md](Medienverteiler-Kurz.md) / [MEDIENVERTEILER-EROEFFNUNG.md](../MEDIENVERTEILER-EROEFFNUNG.md) oder aus Admin → Medienspiegel kopiert (BCC).
- [ ] **Anhänge:** Optional 1–2 Fotos (niedrige Auflösung); sonst keine großen Dateien.

---

## Nach dem Versand

- [ ] **Dokumentation:** In MEDIENVERTEILER-EROEFFNUNG.md unter „Pflege“ Versanddatum + Anlass notieren.
