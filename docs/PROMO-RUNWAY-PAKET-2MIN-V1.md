# Runway-Paket ~2 Minuten (V1)

## Quelle im Code

- **`src/config/promoRunwayPack2min.ts`** – acht Szenen: deutscher Sprechertext, englischer Runway-Bild-Prompt, Screen-Hinweis (ök2-Route).
- **`src/pages/PromoRunwayPackPage.tsx`** – APf-Seite mit Kopieren-Buttons.

## APf-Einstieg

- Route: **`/projects/k2-galerie/promo-runway-pack`** (nur localhost / `?apf=1` / `?dev=1` wie Promo-Video-Produktion).
- Smart Panel → Mappe **Promo-Video-Produktion** → **Runway-Paket ~2 Min**.

## Nutzung in Runway

Runway liest **kein** Repo – Prompts **manuell** einfügen:

1. **Gen-3 Alpha Multi-Shot Video**, Modus **Auto**: gesamten Block **Kombinierter Auto-Prompt (englisch)** kopieren.
2. Oder **pro Szene**: englischen Prompt kopieren; bei 10-s-Clips mehrere Läufe und im Schnitt verbinden.
3. **Echte Screens** parallel: QuickTime auf den angegebenen Routen (siehe Screen-Hinweis pro Szene).

Siehe auch **`docs/PROMO-VIDEO-UMSETZUNG-KONZEPT.md`** (End-to-End: Schnitt, Highlight-Video in Stammdaten).
