# Code 5 – Cursor-Fenster crasht

**Workaround:** App **im normalen Browser** öffnen (z. B. Chrome oder Safari): `http://localhost:5177`  
Dann läuft die App außerhalb der Cursor-Preview – das Cursor-Fenster crasht nicht.

Die Preview (eingebetteter Browser in Cursor) ist anfällig für Abstürze bei schweren Seiten; im normalen Browser ist das kein Thema.
