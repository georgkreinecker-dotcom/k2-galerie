# Stammdaten neu eingeben (nach Datenverlust)

## Was passiert ist

Durch einen Fehler beim „Reparieren“ wurden frühere Stammdaten (Adresse, Willkommensbild) überschrieben. Diese Daten liegen weder im Repo noch in einem auffindbaren Backup – sie müssen daher **einmalig neu eingegeben** werden.

## Was du eingeben musst

- **Adresse der Galerie** (Straße, PLZ, Ort, Land) – im Admin unter Stammdaten → Galerie
- **Willkommensbild** – im Admin unter Stammdaten → Galerie (Bild hochladen)
- **E-Mail und Telefon** – stehen bereits in der Konfiguration (martina.kreinecker@kgm.at, georg.kreinecker@kgm.at, 0664 1046337). Wenn die Felder leer erscheinen, speichere einmal – dann werden sie aus den Defaults gefüllt.

## Künftig: Kein Datenverlust

1. **Regelmäßig „Vollbackup herunterladen“** (im Admin unter Backup & Wiederherstellung). Die Datei z. B. auf **backupmicro** oder im Projektordner ablegen.
2. **„Aus Backup-Datei wiederherstellen“** – wenn du eine solche Backup-Datei hast, kannst du sie hier auswählen; alle Stammdaten, Werke, Events und Dokumente werden daraus wiederhergestellt.
3. **„Aus letztem Backup wiederherstellen“** – nutzt das automatische Backup (alle 5 Sekunden, solange du im Admin arbeitest), falls du nichts heruntergeladen hast.
