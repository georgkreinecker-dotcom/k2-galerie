Musterbilder für die ök2-Galerie (mök2)
========================================

Diese Bilder kannst du in die ök2-Galerie einfügen.

Enthalten:
- willkommen.svg   → Oben auf der Galerie (Menschen / Galerie-Eingang)
- galerie-innen.svg → Unten (Galerie Innenansicht)

So fügst du sie ein:
1. Diese Dateien nach public/img/oeffentlich/ kopieren (gleiche Namen).
2. Die App nutzt sie automatisch für die öffentliche Galerie (ök2).

Für professionelle Fotos: Ersetze die .svg durch .jpg (willkommen.jpg, galerie-innen.jpg) von Unsplash o.ä. und lege sie in public/img/oeffentlich/ ab. Im Code dann .svg durch .jpg ersetzen (tenantConfig.ts → OEK2_WILLKOMMEN_IMAGES).
