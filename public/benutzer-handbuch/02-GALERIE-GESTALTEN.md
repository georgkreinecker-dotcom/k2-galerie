# Galerie gestalten

In diesem Kapitel geht es darum, wie Sie Ihre Galerie mit Leben füllen: Willkommensbild, Werke erfassen, Design anpassen und veröffentlichen.

---

## Willkommensbild und Galerie-Karte

Wenn Besucherinnen und Besucher Ihre Galerie öffnen, sehen sie zuerst oft ein **Willkommensbild**. Das können Sie selbst festlegen.

1. Gehen Sie im Admin in den Tab **„Design“** (oder „Galerie gestalten“).
2. Dort finden Sie den Bereich für das **Willkommensbild** (und ggf. eine Galerie-Karte).
3. Klicken Sie auf **„Bild auswählen“** oder **„Bild übernehmen“**. Sie können ein Foto von Ihrem Gerät wählen oder ein bereits hochgeladenes Bild verwenden.
4. Das Bild wird angezeigt. Wenn Sie zufrieden sind, speichern Sie (Button **„Speichern“** oder **„Übernehmen“**).

Tipp: Ein ruhiges, einladendes Bild wirkt oft besser als ein überladenes. Die App kann das Bild an die Darstellung anpassen.

---

## Werke erfassen und bearbeiten

Ihre Werke (Kunstwerke, Objekte) legen Sie im Admin unter dem Tab **„Werke“** an.

**Neues Werk anlegen:**

1. Klicken Sie auf **„Neues Werk“** (oder „Werk hinzufügen“).
2. Füllen Sie die Felder aus: z. B. Titel, Künstlerin/Künstler, Technik, Jahr, Preis (falls gewünscht). Pflicht ist in der Regel mindestens ein **Foto** und ein **Titel**.
3. Laden Sie ein Foto hoch: Über **„Foto auswählen“** oder **„Bild hinzufügen“** wählen Sie ein Bild von Ihrem Gerät. Die App kann das Bild automatisch freistellen (Hintergrund entfernen), damit das Werk schön zur Geltung kommt.
4. Speichern Sie das Werk.

**Bestehendes Werk bearbeiten:**

1. Öffnen Sie in der Werkliste das gewünschte Werk (Klick auf die Zeile oder auf „Bearbeiten“).
2. Ändern Sie die gewünschten Angaben oder das Bild.
3. Speichern Sie die Änderungen.

Alle Änderungen sind zunächst auf dem Gerät gespeichert. Damit Besucherinnen und Besucher die aktuelle Version sehen, müssen Sie **veröffentlichen** (siehe unten).

---

## Design anpassen

Unter **„Design“** können Sie neben dem Willkommensbild oft auch Texte, Farben oder die Anordnung der Galerie anpassen. Die genauen Optionen hängen von Ihrer Version ab. Wählen Sie den Tab **„Design“** und folgen Sie den Beschreibungen auf der Seite. Nach jeder Änderung lohnt sich ein Klick auf **„Speichern“**.

---

## Veröffentlichen

Damit Ihre Änderungen (neue Werke, neues Willkommensbild, neues Design) für alle sichtbar werden, müssen Sie **veröffentlichen**.

1. Im Admin finden Sie einen Button **„Veröffentlichen“** (oft im Bereich „Einstellungen“ oder oben in der Leiste).
2. Klicken Sie darauf. Die App überträgt die aktuellen Daten.
3. Wenn die Meldung erscheint, dass die Veröffentlichung erfolgreich war, sehen Besucherinnen und Besucher beim nächsten Aufruf der Galerie den neuesten Stand.

Tipp: Veröffentlichen Sie nach größeren Änderungen (z. B. nach dem Anlegen mehrerer Werke), damit auf allen Geräten und für alle der gleiche Stand angezeigt wird.

---

## Kurz zusammengefasst

- **Willkommensbild:** Design-Tab → Bild auswählen/übernehmen → speichern.
- **Werke:** Werke-Tab → Neues Werk → Foto und Angaben eintragen → speichern.
- **Veröffentlichen:** Button „Veröffentlichen“ klicken, damit die neuesten Änderungen für alle sichtbar sind.
