# Häufige Fragen (Vereinsplattform)

Hier finden Sie Antworten auf Fragen, die oft beim Nutzen der Vereinsplattform auftauchen.

---

## Wie aktualisiere ich die Anzeige auf dem Handy?

Wenn Sie am Rechner etwas geändert und veröffentlicht haben, kann es sein, dass Ihr Handy noch die alte Version anzeigt. So bekommen Sie den neuesten Stand:

1. **Seite neu öffnen:** Schließen Sie den Tab oder die App auf dem Handy und öffnen Sie den Link zur Vereinsgalerie erneut.
2. **Oder:** Wenn Sie einen **Stand-Hinweis** sehen (z. B. „Neuer Stand verfügbar“), tippen Sie darauf. Die Seite lädt dann die aktuelle Version.
3. **QR-Code:** Wenn Sie die Plattform über einen QR-Code öffnen, scannen Sie den Code einfach noch einmal – dann wird die neueste Version geladen.

Die App ist so eingerichtet, dass beim Öffnen oder beim Tippen auf den Stand-Hinweis immer die aktuellen Daten vom Server geladen werden.

---

## Werden meine Änderungen am Rechner auch auf dem Handy sichtbar?

Ja – sobald Sie **veröffentlicht** haben. Die Reihenfolge ist:

1. Sie bearbeiten am Rechner (z. B. neues Mitglied, neues Event).
2. Sie klicken auf **„Veröffentlichen“**.
3. Die Daten werden übertragen. Wenn Sie danach die Vereinsgalerie auf dem Handy öffnen (oder die Seite neu laden / den Stand-Hinweis tippen), sehen Sie dort den gleichen Stand wie am Rechner.

Ohne Veröffentlichen bleiben Änderungen nur auf dem Gerät, an dem Sie gearbeitet haben.

---

## Wie drucke ich etwas aus?

Viele Bereiche der App haben einen Button **„Drucken“** oder **„Als PDF drucken“**. Wenn Sie darauf klicken, öffnet sich der Druckdialog Ihres Geräts. Dort können Sie:

- **Direkt drucken:** Drucker wählen und drucken.
- **Als PDF speichern:** Im Druckdialog die Option „Als PDF speichern“ oder „In PDF speichern“ wählen. Die Datei wird auf Ihrem Gerät abgelegt, und Sie können sie später drucken oder weitergeben.

Auf dem Handy kann die Option je nach Gerät etwas anders heißen (z. B. „Teilen“ und dann „Als PDF“). Einmal ausprobieren reicht meist.

---

## Was bedeutet der QR-Code?

Der **QR-Code** ist ein kleines Muster, das Sie mit der Kamera Ihres Handys scannen können. Wenn Sie es tun, öffnet sich automatisch die Vereinsgalerie (oder die gewünschte Seite) im Browser. So können Sie Mitgliedern und Besucherinnen den Code zeigen – sie scannen ihn und landen direkt bei der Vereinsplattform, ohne eine Adresse eintippen zu müssen.

Sie können den QR-Code aus der App heraus anzeigen und – falls vorgesehen – drucken (z. B. für Flyer oder Aushang). Nach jedem **Veröffentlichen** zeigt der gleiche QR-Code die neueste Version; Sie müssen den Code nicht neu drucken, es sei denn, Sie möchten ein neues Layout.

---

## Ich habe mein Passwort vergessen. Was tun?

Das Passwort für den Admin-Bereich wird von Ihrem Verein vergeben. Wenn Sie es vergessen haben, wenden Sie sich an die Person, die Ihnen den Zugang eingerichtet hat (z. B. Vereinsvorstand). Die App speichert das Passwort nicht so, dass Sie es selbst zurücksetzen können – aus Sicherheitsgründen.

---

## Wo finde ich Hilfe, wenn etwas nicht funktioniert?

- **In der App:** Oben oder in der Ecke finden Sie oft einen Link **„Hilfe“** oder **„Handbuch“**. Dort steht diese Anleitung.
- **Drucken:** Sie können diese Seite (oder das gesamte Handbuch) drucken oder als PDF speichern und neben sich legen.
- **Ansprechpartner:** Bei technischen oder inhaltlichen Fragen wenden Sie sich an die Stelle, die Ihnen den Zugang zur Vereinsplattform gegeben hat (z. B. Vorstand, Support).

---

## Kurz zusammengefasst

- **Neuesten Stand sehen:** Seite neu öffnen oder auf „Neuer Stand“ tippen; bei QR-Code einfach noch einmal scannen.
- **Rechner und Handy:** Nach dem Bearbeiten **„Veröffentlichen“** klicken, dann auf dem anderen Gerät die Seite neu laden.
- **Drucken:** Button „Drucken“ in der App → im Dialog Drucker wählen oder „Als PDF speichern“.
- **QR-Code:** Mit Handy-Kamera scannen → Vereinsgalerie öffnet sich; nach Veröffentlichen zeigt er automatisch den neuesten Stand.
- **Passwort vergessen:** An die Stelle wenden, die den Zugang eingerichtet hat (z. B. Vereinsvorstand).
