# Verein, Stammdaten und Mitglieder

## Verein und Stammdaten

Unter **Einstellungen** bzw. **Stammdaten** tragen Sie die Daten Ihres Vereins ein: Name des Vereins, Ansprechpartner, Adresse, Telefon, E-Mail. Diese Angaben erscheinen auf der Vereinsgalerie und auf Drucken (Einladungen, Flyer). Pflegen Sie sie einmal und aktualisieren Sie sie bei Änderungen.

---

## Mitglieder verwalten

Im Bereich **Mitglieder** (oder **Stammdaten – Mitglieder**) legen Sie die Liste der Vereinsmitglieder an. Pro Mitglied können Sie z. B. Name, Kontakt und optional ein Foto erfassen. Die Liste können Sie drucken oder als Adressliste exportieren, wenn die Funktion angeboten wird. So behalten Sie den Überblick und können Einladungen oder Rundschreiben vorbereiten.
